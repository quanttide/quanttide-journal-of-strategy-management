## 核心战略认知

### 竞争力来源

一个组织的竞争力，是 AI 做不了的那部分——把复杂概念讲清楚的能力。AI 能力是花钱就能买到的东西，只有两种差异才是真正的壁垒：
1. **花钱花得更少**——效率优势；
2. **花 token 也干不到的事**——人类独有的认知和判断。

组织要往这两个方向走。

### 从项目制到产品制

公司战略主线不变：从项目制服务转型产品制服务（PaaS 云厂商）。必须做项目养活自己、积累客户和案例，同时自研平台封装日益复杂的业务流程。

目前正处于"服务→产品"的过渡期，PMF 不是找一个原型去验证，而是已经跑通的业务如何规模化。

### PMF 方向判断

各业务的 PMF 形态不同，但存在内在关联：

| 业务 | PMF 方向 | 核心矛盾 |
|------|----------|----------|
| 量潮数据 | 适应客户、建立信任，做好案例库 | 获客能力不足，技术管理尚未形成规模效应 |
| 量潮课堂 | 转向 B 端/团体定制（高校半价、企业全价） | C 端模式不适配，新商业模式未跑通 |
| 量潮咨询 | 服务大客户，小范围接项目积累案例 | 谈判周期长，决策压力大 |
| 量潮云 | 降级为内源项目，统一各业务标准 | 研发投入大，重构进度慢 |

### 获客是核心瓶颈

最大的难点是获客，而获客的难点在于渠道单一和流量限制。各业务线各有解法：
- **数据**：建设公开案例库，让客户看到交付能力
- **咨询**：放弃依赖合作方获客，建立自主获客能力 → 小范围积累案例 → 公开价格和服务 → 让客户上门
- **课堂**：B 端/团体定制 + 两级推荐激励计划 + 转型共建社区
- **云**：通过开源贡献者招募，实际上在招募用户（先用明白才能贡献）

## 各业务线战略决策

### 量潮云 — 开源与共建者策略

**关键决策**：代码已开源，系统可以自生长了。决定向外部贡献者开放。

**三种合作模式探索**：
1. **付费模式** — 按实习生标准付费招募，标准不降
2. **社区模式** — 出让 IP 换合作伙伴（创始人保留态度）
3. **实训模式** — 课程换贡献（最偏好，可控性最强）

**战略逻辑**：借助小红书甲方稀缺的市场状态，把招聘标准放到供应链上，利用公开性博弈。敢于表达诉求，但标准绝不妥协。

**原则**：别人有别人的账，只有遇到我们，我们才能知道合不合适。不需要过多考虑别人怎么样。

### 量潮咨询 — 获客模式重构

**关键决策**：放弃"依赖合作方/盟友获客"模式。

**教训**：没有自主获客能力，就会丧失定价权，可能拿不到项目也分不到钱。

**新策略**：
- 早期小范围接项目 → 积累案例和方案 → 公开价格和服务 → 让客户自己找上门
- 与课堂业务存在天然交叉导流：咨询客户需要课程，课程客户需要咨询

### 量潮数据 — 案例库优先

**关键决策**：客户会议反馈确认，当前第一需求是建设完整的公开案例库。这是接下来一段时间的业务重点。

### 量潮课堂 — 社区化转型

**关键决策**：将课程项目转型为共建社区。这是能发挥公司学习氛围优势的方向。

**配套策略**：
- 两级推荐激励计划：学生推荐他人试课即获学费减免（不需成交，完成试课即可）
- 社群运营需恢复节奏，当前因忙碌和合作伙伴未更新动态而停滞

### 招聘 — 考核分层

**关键决策**：考核必须分层。

**现状判断**：
- 对 T1（中级）最友好
- 对 T0（实训生）和 T2（高级/CTO 助理）都不友好
- 团队需要 CTO 助理（T2），因此考核必须调整
- T0 学员跟第二大脑知识体系困难，内部知识建设尚未成型
- 整体素质在提升，但仍缺二线中层人员

### 品牌 — IP 与新媒体系

**决策**：红苹果作为品牌 IP 吉祥物。无需额外设计，现成可用，计划用于表情包、应用 UI、智能助手。

**设计招标**：计划发布公开征集，寻找长期设计合作伙伴。公开招标流程制度化，同步作为小红书内容素材。

**内容策略**：用内容换数据，以起号为主要目的。利用甲方身份强化互动。

## 认知体系建设

### 认知窗口构想

一个理想化的认知系统：每天只需输入原始信息流，系统自动分类更新。目的是帮助从"一盘大棋"的模糊关联中，提取出标准化的信息结构。

各业务线之间存在内在关联——课堂、数据、云、咨询四种形态，需要以标准化结构存在，才能适应纷繁复杂的市场。

### 公司域结构

将公司分为 N 个域（领域）的方式已验证合理。每天在多方向上的思考，都可以对号入座。

### 系统与人的关系

系统有自己的生命，造物主受系统反馈影响。系统在帮助排除旧错误思想，带来新认知。当前系统已到可以让外部人员接受的程度——目标明确、方向清晰、可以自生长。
